# Projektbrief: Tigube Homepage

## Projektübersicht
Eine professionelle Website für "Tierisch Gut Betreut" - ein Tierbetreuungsservice in Moos. Die Website dient als digitale Visitenkarte und Kundenakquise-Tool für den Tierbetreuungsservice.

## Kernziele
1. **Vertrauen schaffen** - Professionelle Darstellung der Dienstleistungen
2. **Kundengewinnung** - Klare Call-to-Actions für Anfragen
3. **Informationsvermittlung** - Übersichtliche Darstellung aller Services
4. **Lokale Präsenz** - Fokus auf Moos und Umgebung

## Zielgruppe
- Tierbesitzer in Moos
- Berufstätige mit Hunden/Katzen
- Urlauber, die Tierbetreuung benötigen
- Notfallsituationen bei Tierbetreuung

## Kernfunktionen
- **Landingpage** mit Hero-Bereich und Vertrauensindikatoren
- **Services-Übersicht** mit Preisen und Features
- **Über uns** Sektion für Vertrauen und Glaubwürdigkeit
- **Kundenbewertungen** als Social Proof
- **Kontaktformular** für Anfragen
- **Responsive Design** für alle Geräte

## Technische Anforderungen
- Next.js 15 mit React 19
- TypeScript für Typsicherheit
- Tailwind CSS für Styling
- Radix UI für Komponenten
- Responsive Design (Mobile First)
- SEO-optimiert
- Schnelle Ladezeiten

## Erfolgsmetriken
- Conversion Rate (Kontaktanfragen)
- Verweildauer auf der Website
- Mobile Nutzererfahrung
- SEO-Rankings für lokale Suchanfragen
- Bounce Rate

## Abgrenzung
- Keine Online-Buchungsfunktion
- Keine Zahlungsabwicklung
- Keine Benutzerregistrierung
- Fokus auf Informationsvermittlung und Kontaktaufnahme
